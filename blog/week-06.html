About the Class

In this class we learned how to use KiCAD to design our own PCBs and how to mill them. Consequentially, below there are three iterations of the loop station shield for the Raspberry Pi 4, simplifying the circuitry and finally using surface-mount components (SMDs) to miniaturize the form factor.
